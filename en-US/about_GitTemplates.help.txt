TOPIC
    GitTemplates

SHORT DESCRIPTION
    GitTemplates makes finding and generating commonly used files 
    in git repositories easier by using existing templates

LONG DESCRIPTION
    This module is intended to automate some of the common tasks 
    when creating a new git repo. This includes
    creating new .gitignore files from existing templates on the
    github/gitignore repository, generating README.md files that are
    scaffolded from a prompt based cli and the option to inlcude a
    LICENSE.txt file in the process (the more common ones)
